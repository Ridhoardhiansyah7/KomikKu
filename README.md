# App Features
- Release / update terbaru
- Komik Jepang
- Komik Korea
- Komik China
- Bahasa Aplikasi : Inggris, dan Indonesia(Bukan bahasa komiknya)
- Pencarian Komik 
- Genre yang lengkap
- History baca
- Simpan komik untuk dibaca nanti
